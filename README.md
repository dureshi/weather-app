# Weather App

## 📌 Description
A web application that fetches real-time weather data from an external API based on user input.

## 🚀 Features
- Search weather by location
- Real-time API data fetching
- Dynamic UI updates
- Error handling for invalid inputs

## 🛠️ Tech Stack
- JavaScript
- REST API
- HTML/CSS

## 📚 What I Learned
- Fetching data from APIs
- Handling asynchronous operations
- Updating UI dynamically
- Error handling
